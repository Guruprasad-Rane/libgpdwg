gpdwg is python package to read and write `.dwg <https://en.wikipedia.org/wiki/.dwg>`_ files.

It follows dwg file specifications from `Open Design Specification for .dwg files version 5.4.1 <https://www.opendesign.com/files/guestdownloads/OpenDesign_Specification_for_.dwg_files.pdf>`_ prepared by Open Design Alliance.
